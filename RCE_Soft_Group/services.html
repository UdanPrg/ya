Herramientas

Html5

Es un lenguaje markup (de hecho, las siglas de HTML significan Hyper Text Markup Language) usado para estructurar y presentar el contenido para la web.

CSS3

El Lenguaje Que Nos Sirve Para Definir Los Estilos(Colores, Tamaños, Etc.) e Incluso Realizar Animaciones...

JavaScript

El Lenguaje De Programación Para La Web. Desde Simples Calculadoras, Hasta Las Complejidas Que Pudieras Imaginarte.

Bootstrap

Es un framework originalmente creado por Twitter, que permite crear interfaces web con CSS y JavaScript, cuya particularidad es la de adaptar la interfaz del sitio web al tamaño del dispositivo en que se visualice.

Git Hub

Es Una Plataforma de Desarrollo Corporativo para alojar proyectos utilizando el sistema de control de versiones Git.

MongoDB

Base de datos orientada a documentos con esquema dinámico. Esto le permite ofrecer un alto rendimiento y facilita el desarrollo de aplicaciones.

NodeJS

Su meta es permitir a un programador construir aplicaciones altamente escalables y escribir código que maneje Grandes Catidades de Conexiones Simultaneas En Una Máquina Física

jQuery

Es una biblioteca multiplataforma de JavaScript, que permite simplificar la manera de interactuar con los documentos HTML, manipular el árbol DOM, manejar eventos, desarrollar animaciones y agregar interacción con la técnica AJAX a páginas web.
